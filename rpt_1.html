Architecture Decision Option A: Direct API call from bookmarklet (self-contained, needs API key in code) Option B: Local HTML tool page (bookmarklet sends text there, cleaner UX) I'd recommend Option B - the bookmarklet extracts text and opens your local tool, which handles the API call and formats the FYI. Keeps your API key out of the bookmarklet itself.
